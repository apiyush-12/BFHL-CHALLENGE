# BFHL Challenge Frontend

This is the frontend for the SRM Full Stack Engineering Challenge.

## Setup

1. Open a terminal in the `frontend` folder.
2. Run `npm install` to install dependencies.
3. Start the Vite dev server with `npm run dev`.

## Environment Variables

You can set the backend API URL in a `.env` file in the root of the `frontend` folder (optional, defaults to `http://localhost:3000/bfhl`):

```
VITE_API_URL=http://localhost:3000/bfhl
```
